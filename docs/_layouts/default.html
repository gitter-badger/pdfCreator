PDFCreator

A library for creating rich PDFs using JavaScript, it has multiple built-in layouts, and also it provides the ability to create and register custom layouts.

This library is built on top of jspdf to provide abstract methods and custom layouts for building PDF documents, so you will gain both simplicity and reusage of your custom layouts
